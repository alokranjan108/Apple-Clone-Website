# Apple-Clone-Website
# 🍎 Apple Clone Website (Work in Progress)

This project is a simple clone of the Apple website created using **HTML and CSS only**.
It focuses on practicing layout design and styling by recreating the look and feel of Apple's homepage.

> ⚠️ Note: This project is **not fully completed** and **not responsive yet**.

---

## 🚀 Live Demo

👉 https://alokranjan108.github.io/Apple-Clone-Website/

---

## 📌 Current Features

* 🎨 Apple-inspired UI design
* 🧱 Structured layout using HTML
* 🎯 Basic styling with CSS
* 🖥️ Works on desktop view

---

## ⚠️ Limitations

* ❌ Not responsive (may break on mobile/tablet)
* ❌ Some sections are incomplete
* ❌ No JavaScript functionality
* ❌ Limited interactivity

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
